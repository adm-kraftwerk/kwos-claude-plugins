# Changelog — kw-branding

## [1.0.0] — 2026-03-20

### Added
- `/kw-branding:kraftwerk-colouring` — kraftwerk Brand Colors auf beliebige Dateitypen anwenden
  - Unterstuetzte Typen: CSS, SCSS, HTML, JS, TS, SVG, JSON, Markdown, Tailwind, PPTX, DOCX, PowerBuilder
  - Modi: Nur Farben (Standard) | + Headlines | + Sphaeren | Full Branding
  - Design Tokens: CSS, SCSS, JSON (W3C), JS/TS
  - Assets: Logos (SVG), Sphaeren (SVG + Base64), Token-Dateien
  - PPTX/DOCX Script: `scripts/apply_colors.py`
